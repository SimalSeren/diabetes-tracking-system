# Diabetes Tracking System

## Overview
Diabetes Tracking System is a desktop application developed using Python and MySQL.  
The system allows patients to record blood glucose levels, nutrition data, and daily activities, while enabling doctors to monitor and review patient information.

## Features
- Blood glucose tracking
- Nutrition and activity recording
- Doctor–patient interface separation
- Database-backed data storage (MySQL)
- Structured data management

## Technologies Used
- Python
- MySQL
- Tkinter (GUI)
- SQL

## Purpose
This project was developed as a university coursework project to gain experience in database design, desktop application development, and user data management.
